## 1. 背景与动机

近年来，大型语言模型（LLM）在自回归（Auto-Regressive, AR）范式下取得了巨大成功。AR 模型通过严格的从左到右因果依赖建模序列，GPT-4o、LLaMA 3、Qwen3 等模型都基于这一范式。然而，这种逐 token 生成的顺序性带来了根本性的推理瓶颈：无法进行并行化，导致延迟随模型规模增长而急剧增加。同时，严格的因果结构对于需要双向推理和整体理解的任务而言是次优的。

离散掩码扩散语言模型（MDLM, Masked Diffusion Language Model）作为一种有前景的替代方案应运而生。通过从随机掩码输入中重建序列，这些模型天然支持并行生成并充分利用双向上下文。尽管概念优势明确，但目前大多数扩散模型规模较小（≤8B参数），如何将扩散模型扩展到数百亿参数的主流 AR 模型级别，是一个关键前沿挑战。

**LLaDA2.0** 正是在这一背景下提出的——首个将离散扩散语言模型扩展到 **100B 总参数**的模型系列，标志着扩散模型进入了前沿规模部署的新纪元。

## 2. 问题定义

论文旨在解决以下核心问题：

1. **规模鸿沟**: 如何将扩散语言模型从现有的 ≤8B 规模扩展到 100B 级别的参数规模？
2. **训练范式**: 如何避免从头训练扩散模型的巨大成本？
3. **性能对齐**: 扩散模型能否在复杂任务（如代码生成、数学推理、工具调用）上追平甚至超越同规模的 AR 模型？
4. **推理效率**: 如何充分发挥扩散模型并行解码的速度优势，实现实际部署中的效率提升？
5. **训练稳定性**: 在从 AR 到扩散的转换过程中，如何解决数据分布差异和优化不稳定性问题？

## 3. 核心创新

LLaDA2.0 提出了多项关键创新：

### 3.1 两阶段持续预训练（CPT）范式

不同于从头训练扩散模型，LLaDA2.0 以现有 AR 检查点（Ling-mini-2.0 和 Ling-flash-2.0）为基础，通过系统性转换注入扩散能力，保留语言知识的同时引入双向降噪特性。

### 3.2 Warmup-Stable-Decay（WSD）三阶段训练策略

- **Warmup**: 逐步增加块大小（从 1 → 4 → 32 → 64 → 4096），将模型从 AR 模式平滑过渡到全序列 MDLM 模式
- **Stable**: 在 MDLM 范式下进行大规模训练，深化对扩散动态的理解
- **Decay**: 逐步将块大小从 4096 衰减回紧凑规模（如 32），保留全局知识的同时恢复 BDLM 的推理效率优势

### 3.3 文档级注意力掩码（Document-Level Attention Mask）

训练时将异构文档打包成固定长度序列，通过文档级掩码确保注意力仅在单个文档边界内操作，防止跨文档的虚假依赖关系，保证双向建模的语义一致性。

### 3.4 互补掩码策略（Complementary Masking）

在后训练（SFT）阶段，从单个源序列生成两个互补训练实例（主掩码和逆掩码），确保每个 token 位置恰好以未损坏状态呈现一次。这将近 100% 的数据利用率和消除 token 级采样偏差。

### 3.5 置信度感知并行训练（Confidence-Aware Parallel, CAP）

引入辅助置信度损失，仅在正确预测的 token 上最小化输出分布的熵，迫使模型增强其正确预测的确定性。这对实现高效并行解码至关重要。

### 3.6 DPO 适配

将标准 DPO 适配到扩散模型，通过定义基于掩码 token 的序列对数概率，使用 ELBO 替代精确对数似然进行优化，实现有效的偏好对齐。

## 4. 方法论详解

### 4.1 整体训练框架

LLaDA2.0 的训练遵循三阶段递进：

1. **持续预训练（CPT）**: 将 AR 基础模型适配为 MDLM，学习双向降噪
2. **块扩散预训练（Block Diffusion Pre-training）**: 从 token 级过渡到块级扩散建模
3. **后训练（Post-training）**: 包括 SFT 和 DPO，对齐人类意图并优化部署效率

### 4.2 训练目标函数

**BDLM 阶段（Warmup & Decay）**:

$$
\mathcal{L}_{\text{BDLM}}(\theta) = - \mathbb{E}_{t, \bm{x}_0, \bm{x_t}} \left[ \frac{\alpha_{t}'}{1-\alpha_{t}}\sum_{k=1}^{K} \sum_{i=1}^{L_B} \mathbb{1}[x_{t,k}^i=\text{[MASK]}] \log p_{\theta}(\bm{x}_{0,k}^i | \bm{x}_{0,<k}, \bm{x}_{t,k}) \right]
$$

**MDLM 阶段（Stable）**:

$$
\mathcal{L}_{\text{MDLM}}(\theta) = - \mathbb{E}_{t,\bm{x}_0,\bm{x}_t} \left[\frac{\alpha'_t}{1-\alpha_t}\sum_{i=1}^{L}\mathbb{1}[x_t^i=\text{[MASK]}]\log p_{\theta}(x^i_0 | \bm{x}_{t})\right]
$$

**SFT 阶段**:

$$
\mathcal{L}_{\text{SFT}}(\theta) = - \mathbb{E}_{t,(\bm{c},\bm{x}_0),\bm{x}_t} \left[\frac{\alpha_{t}'}{1-\alpha_{t}}\sum^{K}_{k=1}\sum_{i=1}^{L_{B}}\mathbb{1}[x_{t,k}^i=\text{[MASK]}]\log p_{\theta}(x^i_{0,k} |\bm{c}, \bm{x}_{0,<k}, \bm{x}_{t,k})\right]
$$

### 4.3 掩码比率带宽策略

标准离散扩散过程在 [0, 1] 区间采样掩码概率。然而，极值掩码率（接近 0 或 1）引入高梯度方差且信息量极低：

- 接近 0 的掩码率使重建变得过于简单
- 接近 1 的掩码率将目标退化为学习数据边缘分布

LLaDA2.0 将噪声调度裁剪到边界区间 $[\alpha_{\min}, \alpha_{\max}]$，将训练焦点集中在最具信息量的噪声区间，稳定收敛并改善生成困惑度。

### 4.4 Top-k 检查点合并

在完成 BDLM 预训练后，识别性能最优的前 k 个模型检查点，对其参数进行算术平均，形成统一的 BDLM。这有效合成了模型在各种最优训练状态下捕获的多样化"知识"，平滑了参数景观，缓解了过拟合。

### 4.5 推理策略

LLaDA2.0 采用块级推理：

- 每次扩散步骤采样一个块，条件于先前采样的块
- 每个块内部是多步迭代细化过程
- 采用混合接受策略：
  - 首先接受所有采样概率超过置信度阈值（0.95）的 token
  - 若不满足数量要求，触发低置信度回退，接受固定数量的最可能 token
