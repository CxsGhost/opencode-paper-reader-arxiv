# LLaDA2.0-Uni: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model

> **论文标题**: LLaDA2.0-Uni: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model  
> **arXiv ID**: 2604.20796  
> **作者**: Tiwei Bie, Haoxing Chen, Tieyuan Chen, Zhenglin Cheng, Long Cui 等 (Inclusion AI)  
> **发表时间**: 2026年4月

---

## 1. 背景与动机

近年来，大语言模型（LLMs）已从纯文本处理扩展到多模态任务，涵盖视觉理解和图像生成两大核心方向。传统上，这两类任务由独立的专业模型分别处理：例如 Qwen-VL、InternVL 等模型专注于多模态理解，而 Flux、Z-Image 等则专注于图像生成。然而，构建一个能在单一框架内同时处理理解与生成的统一模型，不仅能够促进两类任务之间的相互增强，还能显著提升部署效率，并解锁交错生成与推理等高级能力，从而更接近通用人工智能（AGI）的目标。

当前主流的统一多模态模型大多基于自回归（AR）架构，如 Janus、BAGEL 等通过将图像离散化为 token 序列，在 next-token prediction 的框架下统一处理文本和图像。另一类方法（如 OmniGen2、Hunyuan Image 3.0）则采用文本自回归 + 图像扩散的混合范式。然而，掩码扩散模型（masked diffusion models）作为替代范式，在并行解码和双向上下文建模方面具有天然优势，且可通过单一目标函数简化训练过程。

尽管掩码扩散模型前景广阔，现有统一扩散模型（如 MMaDA、Lumina-DiMOO）在任务覆盖和 benchmark 性能上仍落后于自回归统一架构。这一差距主要源于三个根本性设计缺陷：
1. **重建式 VQ tokenizer 缺乏语义信息**，导致理解性能较差；
2. **VQ tokenizer 对图像过度压缩**，牺牲了生成质量；
3. **完全双向建模对文本建模不可靠**，且现有方法通常假设固定输出长度，限制了开放式场景的应用。

在此背景下，本文提出了 **LLaDA2.0-Uni**，一个基于离散扩散大语言模型（dLLM）的统一多模态理解与生成框架。

---

## 2. 问题定义

LLaDA2.0-Uni 试图解决的核心问题可以概括为：

> **如何设计一个基于掩码扩散范式的统一框架，在单一模型中同时实现与专用模型相媲美的多模态理解能力和高质量的图像生成能力，并原生支持交错生成与推理？**

具体而言，需要克服以下挑战：
- **语义与重建的权衡**：传统的 VQ-VAE tokenizer 专注于像素级重建，丢失了语义信息，不利于理解任务；而纯语义 tokenizer 又缺乏图像重建能力。
- **训练目标的统一**：现有方法常采用自回归与扩散的混合损失，导致优化困难；而纯掩码扩散方法在文本建模上表现不稳定。
- **推理效率**：扩散模型通常需要多步采样，如何在保持质量的同时加速推理是一个关键问题。
- **多任务数据的协同训练**：理解和生成任务的数据分布差异大，如何在统一框架下协同训练而不互相干扰。

## 3. 核心创新

LLaDA2.0-Uni 的主要贡献和创新点包括：

### 3.1 全新统一架构

论文提出了一个由三大核心组件构成的统一架构：
- **SigLIP-VQ 语义离散 Tokenizer**：基于 SigLIP2-g ViT 构建，直接将连续视觉输入转换为离散语义 token，无需像素级重建，从而保留了丰富的语义信息。
- **16B MoE 扩散语言模型（dLLM）骨干**：以 LLaDA-2.0-mini（160亿参数 MoE 架构）为 backbone，通过扩展词表以容纳视觉 token，实现文本和图像在统一空间中的建模。
- **专用扩散解码器**：基于 Z-Image-Base（6B 预训练文本到图像模型）构建，将 dLLM 生成的离散语义 token 作为条件信号，重建高保真图像，并通过蒸馏实现 8 步无 CFG 推理。

### 3.2 交错生成与推理

得益于统一的离散表示，LLaDA2.0-Uni 原生支持文本与图像的交错生成和推理。这是纯自回归或纯扩散模型难以直接实现的能力，标志着理解与生成可以从相互独立走向相互增强。

### 3.3 无需训练的高效推理加速框架 SPRINT

论文提出了 **SPRINT**（Sparse Prefix Retention with Inference-time Non-uniform Token Unmasking），一种无需额外训练的推理加速框架：
- **稀疏前缀保留（Sparse Prefix Retention）**：以模态感知的方式修剪前缀 KV cache，降低每步计算成本。文本 token 保留比例 $r_{\text{text}} = 1.0$，图像 token 保留比例 $r_{\text{img}} = 0.8$，利用图像 token 的空间冗余性实现高效剪枝。
- **非均匀 token 解掩（Non-uniform Token Unmasking）**：用置信度自适应的解掩策略替代固定的去噪 schedule。模型在每个 masked 位置上计算预测置信度，对高于阈值 $\tau$（如 0.93 或 0.95）的位置一次性解掩，实现计算资源的动态分配。

两项技术协同工作，在 backbone 上实现了高达 **1.6x 的加速**，且质量损失可忽略不计。

### 3.4 强大的 Benchmark 性能

实验结果表明，LLaDA2.0-Uni 在多模态理解方面与专用 VLM（如 Qwen2.5-VL-7B）表现相当，在图像生成和理解统一模型中达到 SOTA 水平，同时在图像编辑、交错生成和推理等任务上展现了领先的灵活性和性能。

## 4. 方法论

### 4.1 整体架构

**SigLIP-VQ Tokenizer**

不同于标准 VQ-VAE 依赖像素级重建，SigLIP-VQ 直接在理解任务上训练，保留丰富的语义信息。它使用预训练的 SigLIP2-g ViT 作为视觉特征提取器，支持动态分辨率处理。后续向量量化器将视觉表示与预训练大语言模型对齐，codebook 大小为 16,384，维度为 2,048。

由于 SigLIP-VQ 缺乏原生图像重建机制，论文设计了专用的扩散解码器来解决这一问题。

**MoE dLLM Backbone**

为了整合视觉信息，模型将原始 dLLM 词表扩展，追加 SigLIP-VQ 的 codebook token 以及一组用于图像生成和理解的特殊 token。输入嵌入层保留预训练语言嵌入，新视觉 token 嵌入随机初始化；最终预测头也相应扩展，语言部分从预训练权重初始化以保持语言能力。

为了平衡训练稳定性和性能，论文采用 **Block-wise Attention** 方案。该方案在预定义 block 内限制注意力，并选择性地跨 block 启用注意力，既维持了并行解码的速度优势，又缓解了纯全双向注意力对 SigLIP-VQ token 带来的自回归偏差干扰。

位置编码方面，模型保留 1D RoPE，通过特殊的 `<height>` 和 `<width>` token（如 `<imgsize_512>`）编码 2D 空间信息。这种简单方案已被先前研究证实有效，且支持任意分辨率图像处理。

**Diffusion Decoder**

解码器以 dLLM 生成的图像 token 为**唯一条件信号**，取代了传统的文本 prompt。这与 NextFlow、X-Omni 等方法冗余地组合文本 prompt 与视觉 token 的做法不同。除了基本解码功能，扩散解码器还执行 2x 超分辨率。为了降低 50 步 CFG 采样的计算开销，通过模型蒸馏实现仅需 **8 步的无 CFG 推理**。

### 4.2 训练目标

**BDLM 损失**

训练采用 Block Diffusion Language Model (BDLM) 目标函数，将标准离散扩散扩展到 block 级别的掩码区域：

$$
\mathcal{L}_{\text{BDLM}}(\theta) = - \mathbb{E}_{t, \bm{x}_0, \bm{x_t}} \left[ \frac{\alpha_{t}'}{1-\alpha_{t}}\sum_{k=1}^{K} \sum_{i=1}^{L_B} \mathbb{1}[x_{t,k}^i=\text{[MASK]}] \log p_{\theta}(\bm{x}_{0,k}^i | \bm{x}_{0,<k}, \bm{x}_{t,k}) \right]
$$

其中期望取自时间步 $t$、干净序列 $\bm{x}_0$ 及其加噪版本 $\bm{x}_t$；指示函数 $\mathbb{1}[\cdot]$ 确保仅对 masked token 进行预测；$-\alpha'_t / (1 - \alpha_t)$ 是扩散导出的时间权重。

**MoE 负载均衡**

采用无辅助损失的负载均衡机制（源自 DeepSeek-V3），通过差异化专家专业化同时鼓励均匀工作负载分布。路由门输出缩放因子为 2.5 以稳定 RMS 幅度，偏置使用 RMSNorm 风格归一化更新。

**SFT 优化**

SFT 阶段引入 **Mask Token 重加权损失（MTRS）**：

$$
\mathcal{L}_{\text{MTRS}} = \frac{\sum_{j} \beta_j \mathcal{L}_{\text{SFT}}^{(j)}}{\sum_{j} \beta_j} \,, \quad \text{where} \quad \beta_j = \frac{1}{\sqrt{\sum_{k=1}^{K} \sum_{i=1}^{L_B} \mathbb{1}[x_{t,k}^{i,(j)}=\text{[MASK]}]}}
$$

重加权因子 $\beta_j$ 为样本 $j$ 中 masked token 数量的平方根倒数，平衡不同长度样本的梯度贡献。

此外，采用 **互补掩码（Complementary Masking）** 策略，从单个序列构造两个对偶训练实例，每个 token 位置在配对中恰好出现一次未损坏状态，信息利用率翻倍并消除 token 级采样偏差。

**扩散解码器训练**

整体训练分为三个阶段：
1. **Warm-up**：冻结语义处理器，仅更新其余模块，建立跨模态对齐。
2. **多域泛化**：解冻全部参数，在多样域上微调以增强鲁棒性。
3. **高保真精修**：在高质量数据上精修以提升美学保真度和细粒度视觉细节。

解码器通过轻量级一致性蒸馏框架加速：仅添加一个辅助投影层（推理时丢弃），蒸馏目标结合流匹配损失与一致性项：

$$
\mathcal{L}_{\text{Distill}}(\theta) = \mathbb{E}_{\bm{x}_0, \bm{z}, t} \left[ \| \bm{v}_{\theta,t} - \bm{v}_{t} \|^2_2 + \| \bm{u}_{\theta,t} - \bm{v}_{t} + t \cdot \frac{\text{d}\bm{u}_{\theta^-,t}}{\text{d}t} \|^2_2 \right]
$$

其中时间导数通过二阶差分技术近似，实现 8 步无 CFG 高质量推理。

### 4.3 三阶段训练流水线

| 阶段 | S0: 视觉-语言对齐 | S1: 多任务预训练 | S2: 监督微调 |
|------|-------------------|------------------|-------------|
| 理解数据 | 图像描述、文本 | 图像描述、文本、OCR、定位、计数、视频、多模态 VQA | 高质量多模态 VQA、文本 QA、交错推理 |
| 生成数据 | 文生图 | 文生图、图像编辑、交错生成 | 高质量图像生成、带 CoT 的图像生成、图像编辑、交错生成 |
| 生成分辨率 | 256→512 | 512 | 512（解码器→1024） |
| 理解最大边长 | 800 | 800 | 800 |
| 训练 Token | 100B | 210B | 80B |
| 序列长度 | 8192 | 8192 | 8192→16384 |

### 4.4 数据准备

多模态理解数据涵盖图像描述、OCR（基于 PaddleOCR + Qwen3-VL 精炼）、定位与计数（Objects365、RefCOCO + Qwen3-VL-235B 验证）、世界知识与推理数据，以及来自 Ling2.0 和 LLaDA2.0 的高质量纯文本数据。

图像生成数据从超过 2 亿张网络图像中筛选，经过元数据过滤（分辨率、压缩率）、美学过滤（ArtiMuse 分数>60）和质量过滤（DeQA-Score>4.0），最终保留 1.4 亿张高质量图像，使用 Qwen3-VL-235B-22B 生成描述。

图像编辑数据结合开源数据集（X2Edit、OmniEdit 等）和自动化合成的高质量编辑对。交错数据从 Koala36M 视频语料中严格过滤获得 600 万高质量片段。

## 5. 实验结果

### 5.1 多模态理解

论文在 21 个多模态理解 benchmark 上进行了评估，涵盖通用 VQA、推理和 OCR/文档理解。

与现有基于扩散的统一模型（Lumina-DiMOO、LLaDA-o）相比，LLaDA2.0-Uni 在所有主要类别上均实现显著提升，尤其在通用 VQA（MMStar: 64.1 vs. 58.0）和复杂推理（MMMU: 50.1 vs. 44.9）上。在 OCR 和文档理解等更具挑战性的场景中，LLaDA2.0-Uni 同样表现优异。

最引人注目的是，LLaDA2.0-Uni 与顶级专用 VLM（如 Qwen2.5-VL-7B）表现相当，在 MMStar（64.1 vs. 63.9）和 CountBench（86.0 vs. 84.9）上甚至略有超越。这证实了统一扩散架构已能有效缩小与专用模型之间的差距。

### 5.2 文本到图像生成

**GenEval**：整体得分 0.89，显著超越所有统一模型，在空间排列上获得最高分 Position（0.90）。

**DPG-Bench**：在统一模型中达到 SOTA 的 87.76 分，超越 LLaDA-o（87.04）和 HunyuanImage-3.0（86.10），在 Entity（93.55）和 Other（94.04）子指标上取得最高分。

**OneIG-EN**：得分 0.505，在 Alignment（0.882）和 Reasoning（0.323）上超越所有其他统一模型，但在密集文本生成方面仍有提升空间。

**UniGenBench（EN）**：以 79.63 分创下统一模型新标杆，在 Logic（63.99）和 Layout（90.30）上甚至超越许多专用生成模型。

**CVTG-2K（文本渲染）**：以 0.765 分领先统一模型，在多区域文本生成上表现出卓越稳定性，随着区域数量增加性能下降幅度远小于 BAGEL、Lumina-DiMOO 等基线。

**WISE-Bench（推理感知生成）**：整体得分 0.68，在所有统一模型中排名第一，与专用生成模型 LongCat-Image 持平；引入推理模式后额外提升约 10%。

### 5.3 图像编辑

**ImgEdit**：统一模型中最佳 Overall 得分 3.92，显著优于 OmniGen2（3.44）和 InternVL-U（3.67），在 Adjust 和 Hybrid 任务中表现尤为突出。

**GEdit-Bench**：英文 6.61、中文 6.66，在感知质量类别中表现强劲，证明模型能在执行编辑的同时保持原始图像的视觉质量。

**MICo-Bench（多参考编辑）**：以 47.1 分创下 SOTA，大幅领先 OmniGen2（33.8）和 Qwen-Image（35.9）。同一 dLLM 架构的 Lumina-DiMOO 在此任务上仅获 23.3 分，凸显本文架构和数据 pipeline 的有效性。

### 5.4 交错能力

论文提出了 **InterGen benchmark**，包含 150 个样本，使用 Gemini-3 和 Qwen3-VL 作为评判，从文本连贯性、图文对齐度和 ID 一致性三维度评估。LLaDA2.0-Uni 在 Story Telling 和 Time Series Forecasting 上优于 Emu3.5，在 Explanation 上表现相当。

在交错推理方面，模型成功推导出棋局中的逻辑策略，并提供物理问题的逐步解答，展示了视觉生成与理解协同的初步潜力。

### 5.5 消融实验

**SPRINT 加速**：稀疏前缀保留 + 非均匀解掩的组合在多个 benchmark 上相较基线实现了约 1.6x 加速，且质量损失不显著。

**解码器蒸馏**：8 步蒸馏在保持高质量的同时，大幅降低了推理成本。

## 6. 结论与展望

本文提出的 LLaDA2.0-Uni 是一个基于 dLLM 的统一多模态理解与生成框架，通过 SigLIP-VQ 语义 tokenizer 将视觉输入映射为语义丰富的离散 token，使文本和图像能够在统一空间中被建模。16B MoE dLLM backbone 和专用扩散解码器的组合，实现了端到端的高质量理解与生成。

LLaDA2.0-Uni 在大量 benchmark 上展现了与专用模型相媲美的性能，同时原生支持交错生成和链式推理，体现了统一架构的灵活性和实用性。

未来改进方向包括：
- **增强视觉细节**：SigLIP-VQ tokenizer 在保留细粒度图像细节方面仍有不足，未来可聚焦于更优的重建技术，以提升对细节敏感任务（如图像编辑）的表现。
- **扩展交错能力**：要完全释放复杂交错生成与推理的潜力，需要进一步扩大训练数据规模和模型容量。
- **强化学习优化**：虽然已开始探索 dLLM 的 RL 优化，但其性能优化仍具挑战性，计划在未来版本中进一步完善 RL 框架并开源。

## 7. 个人评价

### 优点

1. **架构简洁且统一**：与采用 ViT+VAE 双路径的 BAGEL、LLaDA-o 等方法不同，LLaDA2.0-Uni 坚持使用**完全离散的语义 token**贯穿理解和生成全流程，避免了异构编码器带来的建模鸿沟和优化目标分歧。这是一个很优雅的设计选择。

2. **语义 tokenizer 的巧妙取舍**：SigLIP-VQ 放弃了像素级重建，专注语义学习，然后通过专用扩散解码器弥补重建能力。这种"分工协作"的思路有效解决了传统 VQ-VAE 语义不足的痛点。

3. **无需训练的推理加速 SPRINT**：在并行解码优势基础上，通过 KV cache 模态感知剪枝和置信度自适应解掩进一步降低成本，最高达 1.6x 加速且损失可忽略，具有很强的工程实用性。

4. **实验验证全面**：覆盖了理解（21 个 benchmark）、生成（GenEval、DPG-Bench、UniGenBench 等）、编辑（ImgEdit、GEdit、MICo）和交错能力（InterGen 自构建 benchmark）四大维度，说服力极强。

5. **原生支持交错推理**：这是统一模型相比专用模型最具差异化的能力之一。论文中棋局策略推导和物理问题求解的定性例子展示了这一方向的巨大潜力。

### 不足

1. **文本渲染仍有提升空间**：在 OneIG-EN 的密集文本生成子任务上表现不及顶尖专用模型，可能是因为语义 tokenizer 对细粒度文字结构的建模能力有限。

2. **训练基础设施依赖较重**：数据 pipeline 高度依赖 Qwen3-VL-235B 进行标注、过滤和精炼，这种对更强教师模型的依赖在一定程度上增加了复现门槛。

3. **推理速度仍有差距**：虽然 SPRINT 实现了 1.6x 加速，但与纯自回归模型相比，扩散范式在推理步骤上仍有数量级差距。如何在保持质量的同时进一步压缩步数，是统一扩散模型面临的核心挑战。

### 总体评价

LLaDA2.0-Uni 是多模态统一建模领域的一项重要工作。它证明了基于掩码扩散的统一架构不仅能匹敌自回归统一模型，在某些场景下甚至能超越专用模型。其"语义离散 token + dLLM + 扩散解码器"的三元架构，以及 SPRINT 加速框架，为下一代统一基础模型提供了一个可扩展、高效且功能丰富的范式。特别是在交错生成与推理这一前沿方向上，LLaDA2.0-Uni 展现了令人鼓舞的初步成果，值得持续关注和跟进。
